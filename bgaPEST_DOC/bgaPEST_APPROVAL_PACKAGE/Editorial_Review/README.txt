Explanation of Editorial Review formats.

The file Editor_transmittal_memo.pdf is the overall review memo from Mike Eberle.

The file bgaPEST_BACK_FROM_editorial_review.pdf shows the "track changes" markup performed in editorial review.
In several cases, it refers to notes: The notes' text is found in bgaPEST_editing_notes.docx

The file bgaPEST_post_ed.pdf contains a few minor comments on reference format.
